# Impressum-Patch

## Reihenfolge

1. In Supabase `sql_impressum_patch.sql` ausführen.
2. Die Dateien aus diesem Paket in GitHub ersetzen bzw. neu anlegen.
3. Committen.
4. Vercel neu deployen lassen.
5. Im Admin-Bereich den Impressum-Text ausfüllen und speichern.

## Neue/angepasste Funktionen

- Kleiner Impressum-Link im Frontend.
- Neue Seite `/impressum`.
- Impressum im Backend editierbar.
- Inhalt wird in Supabase in `app_settings` gespeichert.
